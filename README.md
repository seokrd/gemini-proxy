# Gemini API Proxy

Прокси-сервер для доступа к Google Gemini API из РФ.

## Использование

Замените `https://generativelanguage.googleapis.com` на URL вашего Vercel приложения.

Например:
- Оригинал: `https://generativelanguage.googleapis.com/v1beta/models/imagen-4.0-generate-001:predict`
- Прокси: `https://your-app.vercel.app/v1beta/models/imagen-4.0-generate-001:predict`
